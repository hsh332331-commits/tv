طريقة التشغيل:
1) ثبّت Node.js إصدار 18 أو أحدث.
2) فك الضغط عن الملفات.
3) شغّل start-windows.bat أو افتح PowerShell داخل المجلد واكتب:
   node server.js
4) افتح المتصفح على:
   http://localhost:8080

في حال ظهر أن المنفذ 8080 مستخدم، شغّل بمنفذ مختلف:
PowerShell:
$env:PORT=9090; node server.js

CMD:
set PORT=9090 && node server.js

ملاحظات مهمة:
- الروابط المباشرة قد تتوقف إذا كانت منتهية أو تحتاج اشتراكاً/صلاحية.
- استخدم الموقع فقط مع القنوات والروابط التي تملك حق استخدامها أو لديك ترخيص قانوني لها.
- تشغيل HLS في Chrome/Edge يعتمد على تحميل hls.js من الإنترنت، وتشغيل TS يعتمد على mpegts.js من الإنترنت.
